<h1 align="center">🌍 Global Economic Intelligence Dashboard</h1>

<p align="center">
A data-driven analytics system that analyzes global economic indicators, detects trends, and predicts future economic patterns using data science and machine learning.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
The <b>Global Economic Intelligence Dashboard</b> is a data analytics platform that collects and analyzes
global economic indicators such as GDP, unemployment rate, population, and development metrics.
The system processes historical economic data, detects correlations between indicators,
and visualizes insights through an interactive dashboard.
</p>

<p>
The goal of this project is to demonstrate skills in:
</p>

<hr>

<h2>🎯 Objectives</h2>

<ul>
<li>Collect global economic datasets for multiple countries.</li>
<li>Store structured data in a MySQL database.</li>
<li>Perform statistical analysis on economic indicators.</li>
<li>Analyze long-term economic trends.</li>
<li>Predict future economic indicators using machine learning.</li>
<li>Visualize insights through an interactive dashboard.</li>
</ul>

<hr>
